# about.me

## Installation

Install SASS (http://sass-lang.com/)
	gem install sass

Install Compass (http://compass-style.org/)
	gem install compass

Install Susy (http://susy.oddbird.net/)
	gem install susy

Install rgbapng (https://github.com/aaronrussell/compass-rgbapng)
	gem install compass-rgbapng
